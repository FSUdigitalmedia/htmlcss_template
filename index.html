Beginning HTML and CSS

This is a simple template repo that includes an HTML document and a corresponding stylesheet (CSS). Use these files to complete the following tasks for this assignment:

Clone the Repo

Clone this repo to your local machine using Visual Studio Code.

Make Changes

Click the "Go Live" button in VS Code so you can see the effects of your changes as you make them.

Make some changes to this HTML page. Try to include some tags that we didn't go over in class. You can find out more about HTML on w3schools HTML tutorial. Maybe try to imagine something you'd like to add to the document and find the tags needed to do it. (For example: a table, a button, an emoji, etc.)

Make some changes to the formatting of the page by editing the CSS file. W3schools has section dedicated to CSS.

Submit the Assignment

When you are done, submit the assignment by committing your changes locally and pushing to GitHub. Your assignment is submitted!
